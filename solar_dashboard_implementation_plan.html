핵심 계산 공식
성능비 (PR, Performance Ratio) — 프로젝트의 핵심 지표
발전소가 이론상 낼 수 있는 발전량 대비 실제로 얼마나 냈는가를 보여주는 국제 표준 지표. 오솔라 앱에 이미 "발전 저하율"로 보여주는 바로 그 숫자의 원리입니다.
PR (%) = (실제 발전량 kWh) ÷ (이론 발전량 kWh) × 100 이론 발전량 = 설비 용량(kW) × 일사량(kWh/m²) × 패널 효율(%) 연간 손실액(원) = (1 - PR/100) × 연간 이론발전량 × SMP(원/kWh)
사용자 입력 → 출력 흐름
STEP 1 — 사용자 입력
발전소 기본 정보
발전소 위치(시/도), 설치 용량(kW), 설치 연도, 최근 월별 실제 발전량(kWh) 직접 입력 또는 CSV 업로드
STEP 2 — 자동 계산
이론 발전량 산출
기상청 ASOS API에서 해당 지역 월별 일사량 조회 → 이론 발전량 계산 → PR 산출 → 지역 평균 PR과 비교
STEP 3 — 진단 결과
건강검진 리포트 출력
종합 점수(A~D 등급) / 연간 손실 금액 환산 / 지역 평균 대비 내 발전소 위치 / 권고 액션(점검·세척·부품 교체)
주요 화면 구성 (4개)
🏠 홈 대시보드
발전소 카드 목록, 각 발전소별 PR 게이지, 이상 감지 알림 배지
📊 성능 분석
12개월 실제 vs 이론 발전량 비교 차트, PR 추이, 지역 평균 비교선
💰 수익 손실 계산기
PR 저하로 인한 월별/연간 손실액, SMP 연동 실시간 환산
🩺 건강검진 리포트
A~D 등급 카드, 원인 진단, 권고 액션, PDF 저장(선택)
사용할 공개 API 3개
공공데이터포털
기상청 지상(ASOS) 일자료 조회서비스
전국 주요 도시 일사량(MJ/m²) 데이터. 1904년~현재. 월별 평균값으로 "이론 발전량" 계산에 사용.
data.go.kr 신청REST API / JSON무료
KPX 전력거래소
한국전력거래소 계통한계가격(SMP) API
시간별 SMP(원/kWh) 실시간 조회. URL: openapi.kpx.or.kr. 손실 금액 환산에 사용.
data.go.kr 신청REST API / JSON무료
EPSIS
전력통계정보시스템(EPSIS) 지역별 발전 통계
지역·연도별 태양광 발전량 통계. "지역 평균 PR" 기준값 도출에 사용. CSV 직접 다운로드 가능.
epsis.kpx.or.krCSV 다운로드무료
⚠️ API 신청 유의사항: 기상청 ASOS API는 공공데이터포털(data.go.kr)에서 신청 후 승인까지 최소 1일 소요됩니다. 면접 전날까지는 신청 완료해두세요. 데모 시 실제 API 연동이 안 되면 미리 뽑아둔 CSV 데이터로 대체해도 충분합니다.
API 없이도 가능한 대안 (데모용)
기상청 기상자료개방포털(data.kma.go.kr)에서 지역별 월평균 일사량을 CSV로 직접 다운로드해서 프론트에 내장하면 API 키 없이도 정상 동작합니다. 면접 데모에서는 이 방식이 오히려 더 안정적입니다.

10~15분 설명 스크립트 (권장 흐름)
OPEN (1분)
문제 정의
"오솔라 앱에는 발전 저하율이 13.4%라고 나옵니다. 그런데 사용자 입장에서 이게 많은 건지, 얼마나 손해보고 있는 건지 알 수 없습니다. 저는 이 숫자를 돈으로 번역하고, 무엇을 해야 하는지까지 알려주는 도구를 만들었습니다."
BODY 1 (3분)
데이터 접근 방식 설명
"기상청 ASOS API로 지역별 일사량을, KPX에서 SMP 가격을 가져와서 '이론상 얼마를 발전해야 하는가'를 계산하고, 실제 발전량과 비교했습니다. 이 방식은 태양광 업계의 국제 표준 지표인 PR(성능비)을 기반으로 합니다."
BODY 2 (5분)
데모 시연
실제 발전소 데이터 입력 → PR 산출 → 손실액 확인 → 건강검진 등급 카드 → 권고 액션 순서로 시연. "이 발전소는 지역 평균보다 PR이 8% 낮고, 연간 약 120만원의 손실이 발생하고 있습니다."
CLOSE (2분)
오솔라 로드맵과의 연결
"오솔라가 3분기에 출시할 모니터링 기능과, 삼일PwC와의 협업에서 목표로 하는 '재무와 발전 데이터 결합'이 바로 이 방향입니다. 이 프로토타입은 그 다음 레이어의 첫 번째 화면이 될 수 있다고 생각합니다."
